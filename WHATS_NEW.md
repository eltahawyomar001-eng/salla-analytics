## üî¢ Financial Calculations Explained

### ROI Calculation:
```
ROI = (Total Revenue Gain - Investment) / Investment √ó 100
```

Example:
- Revenue Gain: SAR 1,200,000
- Investment: SAR 53,430
- ROI = (1,200,000 - 53,430) / 53,430 √ó 100 = 2,146%

### Revenue Scenarios:

**Retention Boost:**
```
Gain = Customers_to_Retain √ó Avg_Revenue_per_Customer
```

**Frequency Boost:**
```
Gain = (Avg_Order_Value √ó Frequency_Increase%) √ó Customer_Count √ó Current_Frequency
```

**AOV Boost:**
```
Gain = (Current_AOV √ó AOV_Increase%) √ó Customer_Count √ó Avg_Frequency
```
